Instructions

  • Insert your desired function in the Function box.
  • The graph will be generated automatically.
  • Set X and Y domains properly.
  • Exclude X=0 or negative Xs if they make the function undefined (Ex. Y=log(X) is undefined at X=0 and negative Xs).
  • The Plotter is smart and it activates the options for excluding X=0 and negative Xs if the function includes log(X).
  • Exclude other points if they make the function undefined (Ex. Y=1/(X+1) is undefined at X=-1). Separate numbers by comma.
  • Use parentheses properly to define the function correctly. The statements inside the parentheses are evaluated before other parts of the function. (Ex. Y=(X+2)/3 is not equal to Y=X+(2/3))
  • Use * as multiplication operator.
  • Use ** as power operator.
  • The function is not case-sensitive.
  • You can use these functions:
    • Polynomials (Ex. x**4 + x**3 + x**2 + x + 1)
    • Trigonometric functions: sin(x), cos(x) and tan(x)
    • Hyperbolic functions: sinh(x), cosh(x) and tanh(x)
    • Arc functions: asin(x), acos(x) and atan(x)
    • Logarithm functions: log(x), log10(x)
    • Exponential function: exp(x)
  • You can change the color of graph.
